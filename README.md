# Personalized Recipe Planner

A web-based application that helps users discover recipes, plan meals, and manage ingredients efficiently.  
It provides personalized suggestions based on user preferences.

## Features
- Recipe recommendations
- Meal planning
- Ingredient management

## Tech Stack
- HTML, CSS, JavaScript
